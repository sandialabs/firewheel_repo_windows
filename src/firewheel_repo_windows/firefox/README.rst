###############
windows.firefox
###############

This Model Component adds the `Firefox <https://www.mozilla.org/en-US/firefox/new/>`_ web browser to all VM's decorated with :py:class:`WindowsHost <windows.base_objects.WindowsHost>`.
It uses Firefox version ``66.0.3``.

**Attribute Depends:**
    * ``graph``

**Attribute Provides:**
    * ``firefox``

**Model Component Dependencies:**
    * :ref:`windows.base_objects_mc`
